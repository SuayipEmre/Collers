### Tech && Tools
1. React -  Next.js 
2. TypeScript
3. Tailwind css
4. react-multi-carousel


### Usage
1. Clone the repository: `https://github.com/SuayipEmre/Collers-Task.git`
2. Install dependencies: 
    - `npm install`
3. Start the project: `npm run dev`
